# 新用户引导改版 PRD（示例镜像）

> 这是一份从飞书文档**镜像**到本地的示例，演示 `type: doc` 的格式。采用模板后删除整个 `_example-tenant/`。

## 背景
新用户首启后 7 日留存偏低，集中在第一次核心操作前流失。

## 目标
- 7 日留存 +5pct
- 首次核心操作完成率 +10pct

## 方案要点
1. 三步式引导，突出唯一核心动作。
2. 引导中插入一次"啊哈时刻"演示。

## 风险
- 引导过长导致跳出；需做 A/B。
